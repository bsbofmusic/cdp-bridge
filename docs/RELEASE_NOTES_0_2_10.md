# CDP Bridge 0.2.10

## Highlights

- Added a built-in **Release Notes** entry in both the tray menu and the main window
- Users can jump directly to the GitHub release page for the currently running version
- Makes upgrade history visible inside the app without manual searching

## UI changes

- Tray menu now includes **Open Release Notes**
- Main window developer panel now includes **Open Release Notes**

## Why this matters

Portable builds are often upgraded by replacing the executable directly. This release makes it easy to confirm:

- which version is currently running
- when that version was published
- what changed in that version

directly from the app UI.
